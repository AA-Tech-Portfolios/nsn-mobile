# NSN Project History

## Purpose

This document records meaningful project milestones, product decisions, and documentation changes for NSN.

It is not a full commit log. Git and GitHub remain the source of truth for detailed implementation history.

## Timeline

### 15 August 2021

- Original Bbuds idea captured in Apple Notes.

### 4 May 2026

- NSN GitHub repository created.

### June 2026

#### Product

- Event detail experience refined with clearer sections and view modes.
- Alpha/testing language made more prototype-aware.
- Community Host terminology standardised.

#### Documentation

- Terminology guide added for contributor-facing naming and language decisions. See [terminology.md](terminology.md).
- Naming history clarified across NSN, SoftHello, and SoftShore.

#### Features

- Dedication section added.
- Rare Visitor Cat Easter egg added.

#### Quality

- Tests and checks expanded as alpha polish continued.

## Maintenance Guidance

Update this document only for meaningful milestones, such as:

- Major product direction changes.
- Important naming or terminology decisions.
- Alpha, beta, or public launch milestones.
- Major documentation additions.
- Significant feature introductions.

Do not update it for every small bug fix or commit.
