# NSN (North Shore Nights) 🌙

A social meetup platform focused on meaningful local connections, privacy, and low-pressure social experiences.

Built with:
- Expo
- React Native
- TypeScript
- PNPM

Current features:
- Profile customization
- Photo upload / remove
- Accessibility support
- Meetup discovery
- Privacy-first design

Roadmap:
- Chat system
- Event RSVP
- Weather-aware meetups
- Push notifications
- Web + iOS + Android support
